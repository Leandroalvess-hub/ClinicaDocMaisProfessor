# 🏥 API Clínica Doc+

> 🚀 API desenvolvida para gerenciamento de pequenas clínicas médicas.

---

## 📌 Sobre o Projeto

A **API Clínica Doc+** tem como objetivo facilitar o controle de atendimentos em clínicas de pequeno porte, permitindo o gerenciamento completo de médicos, pacientes e consultas.

---

## ✨ Funcionalidades

### 👨‍⚕️ Médicos
- ➕ Cadastrar médicos  
- 📋 Listar médicos  
- ✏️ Atualizar dados  
- ❌ Remover médicos  

---

### 🧑‍🤝‍🧑 Pacientes
- ➕ Cadastrar pacientes  
- 📋 Listar pacientes  
- ✏️ Atualizar dados  
- ❌ Remover pacientes  

---

### 📅 Consultas
- 🗓️ Agendar consultas  
- 📋 Listar consultas  
- 🔄 Atualizar consultas  
- ❌ Cancelar consultas  

---

### 📢 Atendimento
- 📢 Chamada de pacientes  
- ⏳ Fila de espera  
- ✅ Finalizar atendimento  

---

## 🛠️ Tecnologias Utilizadas

- ⚙️ ASP.NET Core  
- 🗄️ MySQL  
- 🔗 Entity Framework  
- 📦 RESTful API  

---

## 🚀 Como Executar

```bash
# Clonar o repositório
git clone https://github.com/seu-usuario/api-clinica-docplus.git

# Acessar a pasta
cd api-clinica-docplus

# Restaurar dependências
dotnet restore

# Executar o projeto
dotnet run
